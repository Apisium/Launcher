# PureLauncher

An awesome Minecraft Launcher.

## Build

### Requirements

- Nodejs v12.0.0+
- Python v2.7
- A proper C/C++ compiler toolchain
- [CMake](https://cmake.org/)
- [Rust](https://rustup.rs/)

### Commands

```bash
npm i

npm run compile

npm run build
```

## Development

```bash
npm start

npm run run
```
